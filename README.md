# IBM_Data_Science_Course_7_Final_Assignment
The repository contains my completed final assignment for IBM Data Analysis with Python (Course 7) in the IBM Data Science Professional Certificate.
Project Overiew
The goal of this assignment is to analyze housing, automobiles, or other provided datasets using Python, and apply the following techniques:
*Data loading and cleaning
*Exploratory data analysis (EDA)
*Data visualization
*Regression models (Linear, Polynomial, Ridge, ect.)
*Model evaluation 
Tools and Libraries
*Python
*Pandas
*NumPy
*Matplotlib
*Seaborn
*Scikit-Learn
*Jupyter Notebook
Contents
*Final Assignment Notebook (.ipynb)
Contains all analysis, visualization, and model evaluation required for grading.
